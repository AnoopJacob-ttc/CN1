Experiment – 2

CONFIGURE INITIAL SWITCH SETTINGS

← Back to Experiment List ← Back to Main Page

AIM

  -Host Name
  -Banner MOTD
  -Save Configuration Files to NVRAM

TOPOLOGY

HOST NAME

Switch>enable
Switch#configure terminal
Switch(config)#hostname S1
S1(config)#

BANNER MOTD

S1(config)#banner motd "Unauthorized access is strictly prohibited"

SAVE CONFIGURATION FILES TO NVRAM

S1(config)#exit
S1#copy running-config startup-config

SHOW COMMANDS USED FOR VERIFICATION:

S1#show running-config
S1#show startup-config

RESULT:

Configured and verified Host Name, Banner MOTD, and finally save the configuration from RAM to NVRAM
