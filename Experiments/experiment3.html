Experiment – 3

CONFIGURE INITIAL SWITCH SETTINGS

← Back to Experiment List ← Back to Main Page

AIM

  - Verify the Default Switch Configuration
  - Disable DNS lookup
  - Configure domain name
  - Enable Password
  - Enable Secret
  - Save Configuration Files to NVRAM

TOPOLOGY

VERIFY THE DEFAULT SWITCH CONFIGURATION

Switch>enable
Switch#show running-config

DISABLE DNS LOOKUP

Switch#configure terminal
Switch(config)#no ip domain-lookup

CONFIGURE DOMAIN NAME

Switch(config)#ip domain-name nttftrg.com

ENABLE PASSWORD

Switch(config)#enable password cisco

ENABLE SECRET

Switch(config)#enable secret itsasecret

SAVE CONFIGURATION FILES TO NVRAM

Switch#copy running-config startup-config

SHOW COMMANDS USED FOR VERIFICATION:

S1#show running-config
S1#show startup-config

RESULT:

Configured and verified Default Switch Configuration, disable DNS Lookup, configure domain name Enable Password and Enable Secret, and finally save the configuration from RAM to NVRAM
