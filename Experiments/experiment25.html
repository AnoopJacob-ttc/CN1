Experiment – 25

APPLY VLSM TO EFFICIENTLY USE IP ADDRESSES FOR MULTIPLE VLANS

← Back to Experiment List ← Back to Main Page

AIM

  - Given the network 192.168.1.0/24, design a VLSM plan for the following requirements:
  - Determine subnet masks and address ranges for each VLAN.
    - VLAN 10 – 60 hosts
    - VLAN 20 – 25 hosts
    - VLAN 30 – 10 hosts
    - VLAN 40 – 5 hosts
  - Configure the addressing on a Router-on-a-Stick topology using a single router and four VLANs.
  - Verify connectivity between all VLANs.

S1 CONFIGURATION

Switch>enable
Switch#conf t
Switch(config)#no ip domain-lookup
Switch(config)#hostname S1

S1(config)#vlan 10
S1(config-vlan)#vlan 20
S1(config-vlan)#vlan 30
S1(config-vlan)#vlan 40

S1(config-vlan)#interface fa0/1
S1(config-if)#switchport mode access
S1(config-if)#switchport access vlan 10

S1(config-if)#interface fa0/2
S1(config-if)#switchport mode access
S1(config-if)#switchport access vlan 20

S1(config-if)#interface fa0/3
S1(config-if)#switchport mode access
S1(config-if)#switchport access vlan 30

S1(config-if)#interface fa0/4
S1(config-if)#switchport mode access
S1(config-if)#switchport access vlan 40

S1(config-if)#interface GigabitEthernet0/1
S1(config-if)#switchport mode trunk
S1(config-if)#switchport trunk native vlan 99
S1(config-if)#switchport trunk allowed vlan 10,20,30,40

S1(config-if)#end
S1#copy run start

R1 CONFIGURATION

Router>enable
Router#conf t
Router(config)#no ip domain-lookup
Router(config)#hostname R1

R1(config)#interface g0/0/0
R1(config-if)#no shutdown

R1(config-if)#interface g0/0/0.10
R1(config-subif)#encapsulation dot1Q 10
R1(config-subif)#ip address 192.168.1.1 255.255.255.192

R1(config-subif)#interface g0/0/0.20
R1(config-subif)#encapsulation dot1Q 20
R1(config-subif)#ip address 192.168.1.65 255.255.255.224

R1(config-subif)#interface g0/0/0.30
R1(config-subif)#encapsulation dot1Q 30
R1(config-subif)#ip address 192.168.1.97 255.255.255.240

R1(config-subif)#interface g0/0/0.40
R1(config-subif)#encapsulation dot1Q 40
R1(config-subif)#ip address 192.168.1.113 255.255.255.248

R1(config-subif)#end
R1#copy run start
