Experiment – 24

CONFIGURE INTER-VLAN ROUTING USING IPV6 ON A MULTILAYER SWITCH

← Back to Experiment List ← Back to Main Page

AIM

  - Configure Basic Settings
  - Add VLANs to Switch & MLS
  - Configure Layer 3 Switching using IPv6
  - Test Connectivity with Inter-VLAN Routing

TOPOLOGY

VLAN TABLE

VLAN ID VLAN Name IPV6 Network Address Ports to be assigned
10 IT 2001:DB8:ACAD:1::/64 S1: Fa0/1-2
20 HR 2001:DB8:ACAD:2::/64 S1: Fa0/3-4
99 Native S1: G0/1, MLS: G1/0/1

ADDRESSING TABLE

Device Interface IPv6 Address/Prefix Default Gateway Description
MLS VLAN 10 2001:DB8:ACAD:1::1/64 NOT APPLICABLE SVI for VLAN 10
VLAN 20 2001:DB8:ACAD:2::1/64 NOT APPLICABLE SVIV for VLAN 20
IT-PC1 NIC 2001:DB8:ACAD:1::2/64 2001:DB8:ACAD:1::1 Connected to S1 Fa0/1
IT-PC2 NIC 2001:DB8:ACAD:1::3/64 2001:DB8:ACAD:1::1 Connected to S1 Fa0/2
HR-PC1 NIC 2001:DB8:ACAD:2::2/64 2001:DB8:ACAD:2::1 Connected to S1 Fa0/3
HR-PC2 NIC 2001:DB8:ACAD:2::3/64 2001:DB8:ACAD:2::1 Connected to S1 Fa0/4

S1 CONFIGURATION

Switch>enable
Switch#conf t
Switch(config)#hostname S1

S1(config)#vlan 10
S1(config-vlan)#name IT

S1(config-vlan)#vlan 20
S1(config-vlan)#name HR

S1(config-vlan)#vlan 99
S1(config-vlan)#name Native

S1(config-vlan)#interface range fa0/1-2
S1(config-if-range)#switchport mode access
S1(config-if-range)#switchport access vlan 10

S1(config-if-range)#interface range fa0/3-4
S1(config-if-range)#switchport mode access
S1(config-if-range)#switchport access vlan 20

S1(config-if-range)#interface GigabitEthernet0/1
S1(config-if)#switchport mode trunk
S1(config-if)#switchport trunk native vlan 99
S1(config-if)#switchport trunk allowed vlan 10,20,99

S1(config-if)#end
S1#copy run start

MLS CONFIGURATION

Switch>enable
Switch#conf t
Switch(config)#hostname MLS
MLS(config)#ipv6 unicast-routing

MLS(config)#vlan 10
MLS(config-vlan)#name IT
MLS(config-vlan)#vlan 20
MLS(config-vlan)#name HR
MLS(config-vlan)#vlan 99
MLS(config-vlan)#name Native

MLS(config-vlan)#interface g1/0/1
MLS(config-if)#switchport mode trunk
MLS(config-if)#switchport trunk allowed vlan 10,20,99

MLS(config-if)#interface vlan 10
MLS(config-if)#ipv6 address 2001:DB8:ACAD:1::1/64
MLS(config-if)#ipv6 address FE80::1 link-local

MLS(config-if)#interface vlan 20
MLS(config-if)#ipv6 address 2001:DB8:ACAD:2::1/64
MLS(config-if)#ipv6 address FE80::2 link-local

MLS(config-if)#end
MLS#copy run start
