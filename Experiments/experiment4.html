Experiment – 4

CONFIGURE INITIAL SWITCH SETTINGS

← Back to Experiment List ← Back to Main Page

AIM

  - Configure Line Passwords
  - Encrypt all plain text passwords
  - Logging synchronous
  - Exec timeout
  - Save Configuration Files to NVRAM

TOPOLOGY

CONFIGURE LINE PASSWORDS

Switch(config)#line console 0
Switch(config-line)#password cisco
Switch(config-line)#login

Switch(config-line)#line vty 0 15
Switch(config-line)#password cisco
Switch(config-line)#login
Switch(config-line)#exit
Switch(config)#

ENCRYPT ALL PLAIN TEXT PASSWORDS

Switch(config)#service password-encryption

LOGGING SYNCHRONOUS

Switch(config)#line console 0
Switch(config-line)#logging synchronous 

Switch(config-line)#line vty 0 15
Switch(config-line)#logging synchronous

LOGIN ATTEMPT LIMITATION SETTINGS - EXEC TIMEOUT

Switch(config-line)#line console 0
Switch(config-line)#exec-timeout 5

Switch(config-line)#line vty 0 15
Switch(config-line)#exec-timeout 5
Switch(config-line)#end
Switch#

SAVE CONFIGURATION FILES TO NVRAM

Switch#copy running-config startup-config

SHOW COMMANDS USED FOR VERIFICATION:

S1#show running-config
S1#show startup-config

RESULT:

Configure and Verify Line passwords, encrypt all plain text passwords, logging synchronous, Login attempt limitation setting, and finally save the configuration from RAM to NVRAM
