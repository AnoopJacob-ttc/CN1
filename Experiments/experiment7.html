Experiment – 7

CONFIGURE INITIAL ROUTER SETTINGS

← Back to Experiment List ← Back to Main Page

AIM

  - Verify the Default Router Configuration
  - Configure and Verify the Initial Router Configuration
  - Save the Running Configuration File

TOPOLOGY

VERIFY THE DEFAULT ROUTER CONFIGURATION

Router>enable
Router#show running-config

CONFIGURE AND VERIFY THE INITIAL ROUTER CONFIGURATION

PREVENT THE ROUTER FROM TRYING TO RESOLVE INCORRECTLY GIVEN COMMANDS IN THE CLI

Router>enable
Router#configure terminal
Router(config)#no ip domain-lookup

HOST NAME

Router(config)#hostname R1
R1(config)#exit
R1#

SET THE CLOCK

R1#clock set 10:30:15 01 January 2026

BANNER MOTD

R1#configure terminal
R1(config)#banner motd "Unauthorized access is strictly prohibited"

CONFIGURE DOMAIN NAME

R1(config)# ip domain-name nttftrg.com

ENABLE PASSWORD -ENABLE SECRET

R1(config)#enable password cisco
R1(config)#enable secret itsasecret

CONFIGURE LINE PASSWORDS

R1(config)#line console 0
R1(config-line)#password cisco
R1(config-line)#login
R1(config-line)# logging synchronous
R1(config-line)#exec-timeout 5

R1(config-line)#line vty 0 15
R1(config-line)#password cisco
R1(config-line)#login
R1(config-line)# logging synchronous
R1(config-line)#exec-timeout 5
R1(config-line)#exit
R1(config)#

ENCRYPT ALL PLAIN TEXT PASSWORDS

R1(config)#service password-encryption

SAVE CONFIGURATION FILES TO NVRAM

R1(config)#exit
R1#copy running-config startup-config

SHOW COMMANDS USED FOR VERIFICATION

R1#show running-config
R1#show startup-config

RESULT:

Configured and verified initial Router settings
