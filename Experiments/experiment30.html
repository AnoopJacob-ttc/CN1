Experiment – 30

CONFIGURE DHCPV6

← Back to Experiment List ← Back to Main Page

AIM

  - Configure Basic Settings
  - Configure and verify a Stateless DHCPv6 Server
  - Configure and verify a Stateful DHCPv6 Server

TOPOLOGY

ADDRESSING TABLE

Device Interface IPv6 Address / Prefix Default Gateway
R1 G0/0/0 2001:DB8:ACAD:A::1/64 NOT APPLICABLE
G0/0/1 2001:DB8:ACAD:B::1/64 NOT APPLICABLE
S1 VLAN 1 2001:DB8:ACAD:A::2/64
S2 VLAN 1 2001:DB8:ACAD:B::2/64
PC1 NIC Automatic
PC2 NIC Automatic

R1 CONFIGURATION

Router>enable
Router#configure terminal
Router(config)#no ip domain-lookup
Router(config)#hostname R1

R1(config)#ipv6 unicast-routing
R1(config)#interface g0/0/0
R1(config-if)#ipv6 address 2001:DB8:ACAD:A::1/64
R1(config-if)#ipv6 address FE80::A link-local
R1(config-if)#no shutdown

R1(config-if)#interface g0/0/1
R1(config-if)#ipv6 address 2001:DB8:ACAD:B::1/64
R1(config-if)#ipv6 address FE80::B link-local
R1(config-if)#no shutdown

CONFIGURE STATELESS DHCPV6 ON R1

R1(config)#ipv6 dhcp pool STATELESS
R1(config-dhcpv6)#dns-server 2001:8:8::8
R1(config-dhcpv6)#domain-name stateless.com
R1(config-dhcpv6)#exit
R1(config)#interface g0/0/0
R1(config-if)#ipv6 nd other-config-flag
R1(config-if)#ipv6 dhcp server STATELESS

CONFIGURE STATEFUL DHCPV6 ON R1

R1(config-if)#ipv6 dhcp pool STATEFUL
R1(config-dhcpv6)#address prefix 2001:DB8:ACAD:B::/64
R1(config-dhcpv6)#dns-server 2001:8:8::8
R1(config-dhcpv6)#domain-name stateful.com
R1(config-dhcpv6)#exit
R1(config)#interface g0/0/1
R1(config-if)#ipv6 dhcp server STATEFUL
R1(config-if)#ipv6 nd managed-config-flag
R1(config-if)#end
R1#
R1#copy run start

S1 CONFIGURATION

Switch>enable
Switch#configure terminal
Switch(config)#sdm prefer dual-ipv4-and-ipv6 default
Switch(config)#exit
Switch#reload
