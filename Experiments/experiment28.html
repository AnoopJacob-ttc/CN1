Experiment – 28

CONFIGURE STATIC ETHERCHANNEL

← Back to Experiment List ← Back to Main Page

AIM

  - Configure Basic Switch Settings
  - Configure a static EtherChannel (mode on) on both switches
  - Verify Ether-channel Configuration

TOPOLOGY

ADDRESSING TABLE

Device Interface IP Address Subnet Mask
S1 VLAN 1 10.1.1.1 255.255.255.0
S2 VLAN 1 10.1.1.2 255.255.255.0
PC1 VLAN 1 10.1.1.3 255.255.255.0
PC2 VLAN 1 10.1.1.4 255.255.255.0

PAGP ETHERCHANNEL LINK CONFIGURATION TABLE

CONNECTION INTERFACE USED SWITCH MODE PORT-CHANNEL NO.
S1 - S2 Fa0/1, Fa0/2, Fa0/3 S1: on
S2: on		 1

S1 CONFIGURATION

Switch>enable
Switch#conf t
Switch(config)#no ip domain-lookup
Switch(config)#hostname S1

S1(config)#interface vlan 1
S1(config-if)#ip address 10.1.1.1 255.255.255.0
S1(config-if)#description Management VLAN
S1(config-if)#no shutdown

S1(config-if)#interface range fa0/1-3
S1(config-if-range)#channel-group 1 mode on
S1(config-if-range)#end
S1#copy run start

S2 CONFIGURATION

Switch>enable
Switch#conf t
Switch(config)#no ip domain-lookup
Switch(config)#hostname S2

S2(config)#interface vlan 1
S2(config-if)#description Management VLAN 1
S2(config-if)#ip address 10.1.1.2 255.255.255.0
S2(config-if)#no shutdown

S2(config-if)#interface range fa0/1-3
S2(config-if-range)#channel-group 1 mode on

S2(config-if-range)#end
S2#copy run start

PC1 IP CONFIGURATION

IPv4 Address: 10.1.1.3
Subnet Mask: 255.255.255.0

PC2 IP CONFIGURATION

IPv4 Address: 10.1.1.4
Subnet Mask: 255.255.255.0

SHOW COMMANDS USED

S1# show etherchannel summary
S2# show etherchannel summary
S1#Show ip interface brief
S2#Show ip interface brief

RESULT

Configured and verified a static EtherChannel (mode on) on both switches
