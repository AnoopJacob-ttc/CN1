AIM

  - Explore EXEC Modes
  - Set the Clock

TOPOLOGY

Packet Tracer is a network simulation tool used to practice configuring routers, switches, and networks without using real hardware.

WHY DO WE USE PACKET TRACER?

To learn networking practically
To design and test networks
To practice Cisco commands safely
To prepare for CCNA labs

BASIC COMPONENTS IN PACKET TRACER

End Devices: PC, Laptop, Server, Printer
Network Devices: Router, Switch, Firewall, Wireless Router
Connections: Copper Cable, Fiber, Console Cable
Workspace: Area where you build the network
CLI Tab: To enter IOS commands

EXPLORE EXEC MODES

Switch>				//User Exec Mode
Switch>enable
Switch#				// Privileged Exec Mode
Switch# configure terminal
Switch(config)#			//Global Configuration Mode
Switch(config)#interface fa0/1
Switch(config-if)#		//Interface Mode (Specific Configuration Mode)

SET THE CLOCK

Switch>enable
Switch#clock set 10:30:15 01 January 2026
Switch#show clock

RESULT:

Configured and verified Clock and explored IOS Exec Modes
