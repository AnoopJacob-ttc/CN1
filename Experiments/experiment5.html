Experiment – 5

IMPLEMENT BASIC CONNECTIVITY

← Back to Experiment List ← Back to Main Page

AIM

  - Perform Basic Configurations on a Switch
  - Configure the PCs
  - Configure the Switch Management Interface

TOPOLOGY

ADDRESSING TABLE

Device Interface IP Address Subnet Mask
S1 VLAN 1 192.168.1.253 255.255.255.0
S2 VLAN 1 192.168.1.254 255.255.255.0
PC1 NIC 192.168.1.1 255.255.255.0
PC2 NIC 192.168.1.2 255.255.255.0

PERFORM BASIC CONFIGURATIONS ON A SWITCH

PERFORM BASIC CONFIGURATIONS ON S1

Switch>enable
Switch#configure terminal
Switch(config)#no ip domain-lookup
Switch(config)#hostname S1
S1(config)#ip domain-name nttftrg.com
S1(config)#enable secret class
S1(config)#banner motd "Unauthorized access is strictly prohibited"
S1(config)#service password-encryption
S1(config)#no ip domain-lookup
S1(config)#line console 0
S1(config-line)#password cisco
S1(config-line)#login
S1(config-line)#logging synchronous
S1(config-line)#exec-timeout 5
S1(config)#line vty 0 15
S1(config-line)#password cisco
S1(config-line)#login
S1(config-line)#logging synchronous
S1(config-line)#exec-timeout 5
S1(config-line)#exit
S1(config)#

PERFORM BASIC CONFIGURATIONS ON S2

Switch>enable
Switch#configure terminal
Switch(config)#no ip domain-lookup
Switch(config)#hostname S2
S2(config)#ip domain-name nttftrg.com
S2(config)#enable secret class
S2(config)#banner motd "Unauthorized access is strictly prohibited"
S2(config)#service password-encryption
S2(config)#no ip domain-lookup
S2(config)#line console 0
S2(config-line)#password cisco
S2(config-line)#login
S2(config-line)#logging synchronous
S2(config-line)#exec-timeout 5
S2(config)#line vty 0 15
S2(config-line)#password cisco
S2(config-line)#login
S2(config-line)#logging synchronous
S2(config-line)#exec-timeout 5
S2(config-line)#exit
S2(config)#

CONFIGURE THE PCs

PC1

IP Address: 192.168.1.1
Subnet Mask: 255.255.255.0

PC2

IP Address: 192.168.1.2
Subnet Mask: 255.255.255.0

CONFIGURE THE SWITCH MANAGEMENT INTERFACE

S1#configure terminal
S1(config)#interface vlan 1
S1(config-if)#ip address 192.168.1.253 255.255.255.0
S1(config-if)#description Management Interface
S1(config-if)#no shutdown
S1(config-if)#end
S1#

SAVE CONFIGURATION FILE TO NVRAM

S1#copy running-config startup-config

S2 CONFIGURATION

S2(config)#interface vlan 1
S2(config-if)#ip address 192.168.1.254 255.255.255.0
S2(config-if)#description Management Interface
S2(config-if)#no shutdown
S2(config-if)#end
S2#

SAVE CONFIGURATION FILE TO NVRAM

S2#copy running-config startup-config

SHOW COMMANDS USED FOR VERIFICATION:

S1#show running-config
S1#show startup-config
S1#show ip interface brief
S1#show interface vlan 1
C:\>ping 192.168.1.1

RESULT:

Configured and verified Basic Configuration on Switches, Configure the PCs, Configure the Switch Management Interface, save the configuration from RAM to NVRAM, and finally tested End to End Connectivity.
