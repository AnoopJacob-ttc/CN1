Experiment – 22

CONFIGURE ROUTER-ON-A-STICK INTER-VLAN USING IPV6 ADDRESSES

← Back to Experiment List ← Back to Main Page

AIM

  - Configure Basic Settings
  - Add VLANs to a Switch
  - Test Connectivity with Inter-VLAN Routing
  - Configure Sub interfaces

TOPOLOGY

VLAN TABLE

VLAN ID VLAN Name IPV6 Network Address Ports to be assigned
10 IT 2001:DB8:ACAD:1::/64 S1: Fa0/1-2
20 HR 2001:DB8:ACAD:2::/64 S1: Fa0/3-4
99 Native S1: G0/1

ADDRESSING TABLE

Device Interface IPv6 Address/Prefix Subnet Mask Description
R1 G0/0/0.10 2001:DB8:ACAD:1::1/64 NOT APPLICABLE Sub-Interface for VLAN 10
G0/0/0.20 2001:DB8:ACAD:2::1/64 NOT APPLICABLE Sub-Interface for VLAN 20
G0/0/0.99 No IPv6 Address
IT-PC1 NIC 2001:DB8:ACAD:1::2/64 FE80::1 Connected to S1 Fa0/1
IT-PC2 NIC 2001:DB8:ACAD:1::3/64 FE80::1 Connected to S1 Fa0/2
HR-PC1 NIC 2001:DB8:ACAD:2::2/64 FE80::1 Connected to S1 Fa0/3
HR-PC2 NIC 2001:DB8:ACAD:2::3/64 FE80::1 Connected to S1 Fa0/4

S1 CONFIGURATION

Switch>enable
Switch#conf t
Switch(config)#hostname S1

S1(config)#vlan 10
S1(config-vlan)#name IT

S1(config-vlan)#vlan 20
S1(config-vlan)#name HR

S1(config-vlan)#vlan 99
S1(config-vlan)#name Native

S1(config-vlan)#interface range fa0/1-2
S1(config-if-range)#switchport mode access
S1(config-if-range)#switchport access vlan 10

S1(config-if-range)#interface range fa0/3-4
S1(config-if-range)#switchport mode access
S1(config-if-range)#switchport access vlan 20

S1(config-if-range)#interface GigabitEthernet0/1
S1(config-if)#switchport mode trunk
S1(config-if)#switchport trunk native vlan 99

S1(config-if)#end
S1#copy run start

R1 CONFIGURATION

Router>enable
Router#conf t
Router(config)#hostname R1
R1(config)#ipv6 unicast-routing

R1(config)#interface g0/0/0
R1(config-if)#no shutdown

R1(config)#interface g0/0/0.10
R1(config-subif)#encapsulation dot1Q 10
R1(config-subif)#ipv6 address 2001:DB8:ACAD:1::1/64
R1(config-subif)#ipv6 address FE80::1 link-local

R1(config)#interface g0/0/0.20
R1(config-subif)#encapsulation dot1Q 20
R1(config-subif)#ipv6 address 2001:DB8:ACAD:2::1/64
R1(config-subif)#ipv6 address FE80::1 link-local

R1(config)#interface g0/0/0.99
R1(config-subif)#encapsulation dot1Q 99 Native

R1(config-if)#end
R1#copy run start
