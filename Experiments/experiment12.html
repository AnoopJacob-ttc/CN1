Experiment – 12

CONNECT A ROUTER TO A LAN USING IPV6 ADDRESSING

← Back to Experiment List ← Back to Main Page

AIM

  - Display Router Information
  - Configure Basic Settings
  - Configure Router Interfaces
  - Verify the Configuration

TOPOLOGY

ADDRESSING TABLE

Device Interface IPv6 Address / Prefix Default Gateway
R1 G0/0/0 2001:DB8:ACAD:A::1/64 NOT APPLICABLE
G0/0/1 2001:DB8:ACAD:B::1/64 NOT APPLICABLE
S1 VLAN 1 2001:DB8:ACAD:A::2/64
S2 VLAN 1 2001:DB8:ACAD:B::2/64
PC1 NIC 2001:DB8:ACAD:A::F/64 FE80::1
PC2 NIC Automatic

DISPLAY ROUTER INFORMATION

PERFORM BASIC CONFIGURATIONS ON S1

Router>enable
Router#show running-config

CONFIGURE BASIC SETTINGS

Router>enable
Router#clock set 10:30:15 01 January 2026
Router#configure terminal
Router(config)#no ip domain-lookup
Router(config)#hostname R1
R1(config)#banner motd "Unauthorized access is strictly prohibited"
R1(config)# ip domain-name nttftrg.com
R1(config)#enable password cisco
R1(config)#enable secret itsasecret
R1(config)#service password-encryption

R1(config)#line console 0
R1(config-line)#password cisco
R1(config-line)#login
R1(config-line)# logging synchronous
R1(config-line)#exec-timeout 5

R1(config-line)#line vty 0 15
R1(config-line)#password cisco
R1(config-line)#login
R1(config-line)# logging synchronous
R1(config-line)#exec-timeout 5
R1(config-line)#exit

CONFIGURE ROUTER INTERFACES

R1(config)#ipv6 unicast-routing
R1(config)#interface g0/0/0
R1(config-if)#ipv6 address 2001:DB8:ACAD:A::1/64
R1(config-if)#ipv6 address FE80::1 link-local
R1(config-if)#no shutdown

R1(config-if)#interface g0/0/1
R1(config-if)#ipv6 address 2001:DB8:ACAD:B::1/64
R1(config-if)#ipv6 address FE80::1 link-local
R1(config-if)#no shutdown
R1(config-if)#end

R1#copy run start

S1 CONFIGURATION

Switch>enable
Switch#configure terminal
Switch(config)#sdm prefer dual-ipv4-and-ipv6 default
Switch(config)#exit
Switch#reload

System configuration has been modified. Save? [yes/no]:yes
Proceed with reload? [confirm][Press Enter Here]

[After reload]
Switch>enable
Switch#conf t
Switch(config)#hostname S1

S1(config)#interface vlan 1
S1(config-if)#ipv6 address 2001:DB8:ACAD:A::2/64
S1(config-if)#ipv6 address FE80::1 link-local
S1(config-if)#no shutdown
S1(config-if)#end

S1#copy run start

S2 CONFIGURATION

Switch>enable
Switch#conf t
Switch(config)#sdm prefer dual-ipv4-and-ipv6 default

Switch(config)#exit
Switch#reload

System configuration has been modified. Save? [yes/no]:yes
Proceed with reload? [confirm][press Enter Here]

[After reload]
Switch>enable
Switch#conf t
Switch(config)#hostname S2

S2(config)#interface vlan 1
S2(config-if)#ipv6 address 2001:DB8:ACAD:B::2/64
S2(config-if)#ipv6 address FE80::1 link-local
S2(config-if)#no shutdown
S2(config-if)#end
S2#copy run start

PC1 IPV6 CONFIGURATION

IPv6 Address: 2001:DB8:ACAD:A::F/64
Default Gateway: FE80::1

PC2 IPV6 CONFIGURATION

IPv6 Configuration: Automatic

SHOW COMMANDS USED FOR VERIFICATION

R1#show running-config
R1#show ip interface brief
S1#show ip interface brief
S2#show ip interface brief
C:/>ipconfig /all
All devices will be able to ping each other using IPv6 Address.

RESULT
